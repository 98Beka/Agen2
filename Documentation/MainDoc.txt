# Основная документация проекта

## Технический стек
- **Backend**: FastAPI
- **Frontend**: Next.js
- **База данных**: Qdrant
- **ИИ**: OpenAI API

## Функциональные модули

### 1. Интерфейс чата
  - Пользовательский интерфейс
  - Текстовые беседы

### 2. Система исправлений
  - Обнаружение ошибок
  - Объяснение исправлений

### 3. Словарный модуль
  - Подбор синонимов
  - Примеры использования

### 4. Система прогресса
  - Отслеживание результатов
  - Аналитика улучшений

### 5. Образовательные модули
- **Грамматика**:
  - Уроки по уровням
  - Практические задания
- **Словарный запас**:
  - Расширение запаса
  - Интерактивные игры
- **Чтение**:
  - Тексты по уровням
  - Задания на понимание
- **Устная речь**:
  - Произношение
  - Обратная связь

## План развертывания
- Контейнеризация (Docker)
- Серверная часть с GPU
