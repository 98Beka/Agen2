# Документация по фронтенду

### 1. Главная страница
- **Описание**: Первая страница при входе в систему
- **Содержание**:
  - Краткое описание проекта
  - Блок "Прогресс и ошибки":
    - Результаты пользователя
    - Список ошибок с фильтрацией
    - Аналитика прогресса

### 2. Сайдбар
- **Расположение**: Левая часть страницы
- **Компоненты**:
  - Кнопка "Главное меню"
  - Список доступных уроков

### 3. Страница урока
- **Активация**: При выборе урока в сайдбаре
- **Структура**:
  - Верхние вкладки:
    - Грамматика
    - Словарный запас
    - Чтение
    - Общение

### 4. Обучающие модули
- **Грамматика**:
  - Уроки по уровням
  - Доступ к теории
- **Словарный запас**:
  - Интерактивные задания
  - Доступ к теории
- **Чтение**:
  - Практические задания
  - Доступ к теории
- **Произношение**:
  - Упражнения с распознаванием
  - Доступ к теории
